How the tour actually works in product: dim-everything-except-spotlight layer mounts on first-visit (read once from localStorage). The spotlight cutout animates to the target element. A small tooltip floats next to it with title + body + a row of progress dots + Skip / Next. Clicking Next moves the spotlight to the next target (page tabs → layer lenses → friction zones), tooltip text changes accordingly. Clicking Skip tour dismisses everything and writes pulse_tour_seen: true to localStorage. Tour never re-shows for that browser. Esc also dismisses. Pattern reference: Linear's first-run, Stripe Dashboard's tour, Figma's onboarding.

Static mockup caveat noted — these are visual references only, not interactive. Once approved as direction, the interactivity gets built in the Next.js app.
